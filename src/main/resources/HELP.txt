connect
	Команда для подключения к соответствующей БД
	Формат команды: connect|username|password
	где: 
		username -  имя пользователя БД (совпвдает с именем БД)
		password - пароль пользователя БД
	Формат вывода: текстовое сообщение с результатом выполнения операции.
tables
	Команда выводит список всех пользовательских таблиц 
		содержащихся в БД к которой вы подключены.
	Формат команды: tables
	Формат вывода: табличка в консольном формате	
columns
	Команда выводит список всех колонок  
		содержащихся в запрашиваемой таблице.
	Формат команды: columns|tableName
	где tableName - имя запрашиваемой таблицы
	Формат вывода: табличка в консольном формате	
tabletype
	Команда выводит список всех колонок, 
					тип данных для соответсвующей колонки, 
					возможность содержать null значение соответсвующей колонки
		в запрашиваемой таблице.
	Формат команды: tabletype|tableName
	где tableName - имя запрашиваемой таблицы
	Формат вывода: табличка в консольном формате
columntype
	Формат команды: columntype|tableName|columnName
	где tableName - имя запрашиваемой таблицы
		colummnName - имя столбца в запрашиваемой таблице
	Команда выводит  колоноку, 
					тип данных для соответсвующей колонки, 
					возможность содержать null значение соответсвующей колонки
		в запрашиваемой таблице.
	Формат вывода: табличка в консольном формате
clear
	Команда очищает содержимое указанной (всей) таблицы
	Формат команды: clear|tableName
	где tableName - имя очищаемой таблицы
	Формат вывода: текстовое сообщение с результатом выполнения операции
drop
	Команда удаляет заданную таблицу
	Формат команды: drop|tableName
	где tableName - имя удаляемой таблицы
	Формат вывода: текстовое сообщение с результатом выполнения операции
create
	Команда создает новую таблицу с заданными полями
	Формат команды: create|tableName|column1(data type(data size)) nullable|column2(data type(data size)) nullable|...
	                                                                           |columnN(data type(data size)) nullable
	где: tableName - имя таблицы
	column1 - имя первого столбца записи
		(data type(data size)) - (тип данных колонки(максимальный размер данных для колонки))
			nullable:
				null - при создании записи в столбце значение может содержать null
				not null - при создании записи в столбце значение НЕ может содержать null
	column2 - имя второго столбца записи
		-//-
	columnN - имя n-го столбца записи
		-//-
	Формат вывода: текстовое сообщение с результатом выполнения операции
find 
	Команда для получения содержимого указанной таблицы
	Формат команды: find|tableName
	где tableName - имя таблицы
	Формат вывода: табличка в консольном формате
filetable
	Команда для сохранения содержимого указанной таблицы в файл
	Формат команды: filetable|tableName
	где tableName - имя таблицы
	Формат вывода: табличка в консольном формате + текстовое сообщение с результатом выполнения операции
findsettings
	Команда для получения содержимого указанной таблицы по определенным критериям
	Формат команды: find|tableName|columnName|value|column2|value2|...|columnN|valueN
	где tableName - имя таблицы
		column1 - имя первого столбца записи
		value1 - значение первого столбца записи
		column2 - имя второго столбца записи
		value2 - значение второго столбца записи
		columnN - имя n-го столбца записи
	Формат вывода: табличка в консольном формате
insert
	Команда для вставки одной строки в заданную таблицу
	Формат команды: insert|tableName|column1|value1|column2|value2|...|columnN|valueN
	где: tableName - имя таблицы
	column1 - имя первого столбца записи
	value1 - значение первого столбца записи
	column2 - имя второго столбца записи
	value2 - значение второго столбца записи
	columnN - имя n-го столбца записи
	valueN - значение n-го столбца записи
	Формат вывода: текстовое сообщение с результатом выполнения операции
update
	Команда обновит запись, установив значение column2 = value2, для которой соблюдается условие column1 = value1
	Формат команды: update|tableName|column1|value1|column2|value2
	где: tableName - имя таблицы
	column1 - имя столбца записи которое проверяется
	value1 - значение которому должен соответствовать столбец column1 для обновляемой записи
	column2 - имя обновляемого столбца записи
	value2 - значение обновляемого столбца записи
	columnN - имя n-го обновляемого столбца записи
	valueN - значение n-го обновляемого столбца записи
	Формат вывода: табличный, как при find со старыми значениями обновленных записей.
delete
	Команда удаляет одну или несколько записей для которых соблюдается условие column = value
	Формат команды: delete|tableName|column|value
	где: tableName - имя таблицы
	Column - имя столбца записи которое проверяется
	value - значение которому должен соответствовать столбец column1 для удаляемой записи
	Формат вывода: табличный, как при find со старыми значениями удаляемых записей.
readQuery
	Команда для ввода SQL запроса
	Формат команды: readQuery|SQLQuery (Только чтение из БД)
	где: SQLQuery - ваш SQL запрос
	Формат вывода: табличка в консольном формате
cudQuery	
	Формат команды:	cudQuery|SQLQuery (Для внесения изменений в таблицу)
	где: SQLQuery - ваш SQL запрос
	Формат вывода: текстовое сообщение с результатом выполнения операции		
history
	Команда для получения истории работы с приложением
	Формат команды: history
	Формат вывода: дата и время использования команды -> действия
							результат выполнения команды
help 
	Команда выводит в консоль список всех доступных команд
	Формат команды: help 
	Формат вывода: текст, описания команд
exit 
	Команда для отключения от БД и выход из приложения
	Формат команды: exit
	Формат вывода: текстовое сообщение с результатом выполнения операции
